# High Revenue Order Prediction

## Overview
This project builds a classification model to predict high-revenue orders using product, pricing, and sales channel data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Key Insights
- Order quantity is the strongest driver of high revenue
- Wholesale channel contributes most to high-value orders
- Discounts negatively impact revenue performance
- Pricing alone is not a strong predictor of high revenue

## Files
- HighRevenuePredictionAnalysis.ipynb → full analysis notebook

## How to Run
1. Open the notebook in Jupyter or Google Colab  
2. Run all cells from top to bottom
